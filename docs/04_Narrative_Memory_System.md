# [기획서 04] 지능형 서사 생성 및 장기 메모리 관리 시스템

## 1. 서사 전개 알고리즘
- **Pacing Control:** AI가 현재 서사의 긴장도를 측정. 너무 루즈해지면 '우연한 사건'이나 '갈등 NPC'를 난입시켜 몰입도 유지.
- **Consistency Checker:** Gemini 3의 추론 모드를 별도 실행하여, 새로 생성된 문장이 이전 설정(예: 주인공의 알레르기 설정 등)과 충돌하는지 0.1초 내 검토.

## 2. 지능형 기억(Memory) 계층
1. **Short-term (최근 10턴):** 대화의 흐름과 뉘앙스를 위한 원본 텍스트 유지.
2. **Mid-term (현재 에피소드):** 현재 챕터에서 일어난 핵심 사건들을 요약 리스트로 관리.
3. **Long-term (세계관/캐릭터):** 유저가 입력한 초기 설정 및 게임 중 확정된 중요한 과거사(핵심 로어).

## 3. 분기 및 세이브 시스템
- **State Snapshot:** 모든 대화 턴마다 스탯, 현재 위치, 캐릭터 호감도 상태를 스냅샷으로 저장하여 '타임라인 되돌리기' 기능 완벽 지원.

# [기획서 03] 실시간 비주얼 및 멀티모달 연출 오케스트레이션

## 1. 캐릭터 이미지 트리거 (Sprite Management)
- **감정 ID 연동:** AI가 판단한 `mood_score`와 `metadata`를 기반으로 캐릭터 스프라이트 시트 전환.
- **의상/장비 반영:** 소설 전개 중 캐릭터가 옷을 갈아입거나 부상을 입으면, 해당 상태 태그를 이미지 경로에 조합 (ex: `char_01_casual_injured.png`).

## 2. 환경 및 배경 연출 (Environmental FX)
- **시간/날씨 동기화:** 나레이션에서 "밤", "비" 키워드 추출 시 프론트엔드 CSS 필터(Brightness 0.5, Blue tint) 및 빗줄기 파티클 효과 자동 활성화.
- **카메라 연출:** 긴박한 전투 대사 출력 시 '화면 흔들림(Shake)', 중요 고백 장면 시 '줌인(Zoom-in)' 효과를 AI 응답의 `pacing` 값에 따라 트리거.

## 3. Gemini 3.1 Flash Image 연동 (고급)
- **실시간 삽화 생성:** 특정 중요 분기점 도달 시, 현재 상황 텍스트를 기반으로 Gemini 3.1 Flash Image API를 호출하여 즉석에서 고퀄리티 배경 삽화 생성 및 로딩.

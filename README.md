# 🎭 Project Infinite Persona

> **A Next-generation Web Novel Engine powered by Gemini 3 Flash: Real-time dynamic story & persona generation.**

---

## 📖 About This Game
**Infinite Persona**는 정해진 시나리오의 한계를 넘어, 플레이어와 AI가 함께 실시간으로 서사를 창조하는 **차세대 AI 기반 동적 비주얼 노벨 엔진**입니다. 

기존의 미연시나 비주얼 노벨이 선택지에 따른 고정된 분기를 제공했다면, 본 프로젝트는 **Gemini 3 Flash**의 강력한 추론 능력을 활용하여 유저가 입력한 설정에 따라 세상에 단 하나뿐인 이야기를 실시간으로 생성합니다.

### 🌟 핵심 컨셉 (Core Concepts)
* **무한한 페르소나:** 유저가 직접 정의하는 캐릭터의 '영혼'과 말투.
* **실시간 서사 생성:** 스크립트 없는 자유로운 소설적 전개.
* **시각적 오케스트레이션:** 감정 분석을 통한 표정 및 배경 실시간 연동.

---

## ✨ Key Features
- **Persona Engine:** High-fidelity character voices using Gemini 3.1.
- **Visual Triggers:** Automatic expression and background changes via JSON metadata.
- **Persistent Memory:** Long-term world-building with Gemini's massive context window.

## 🛠 Tech Stack
- **Frontend:** Next.js 15 (App Router)
- **Backend:** FastAPI (Python), LangChain
- **AI Model:** Google Gemini 3 Flash

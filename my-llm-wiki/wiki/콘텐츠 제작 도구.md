# 콘텐츠 제작 도구
## Summary
프로젝트 안에는 세계관과 캐릭터를 직접 편집하는 패널이 들어 있다. 실제 저장 대상은 `StoryProfileStore`의 `user://content/*.json`이며, 이미지 업로드는 공통 크롭 팝업을 통해 정규화된다.

## Key Points
- `StoryBuilderPanel`은 현재 메인 메뉴에서 쓰는 세계관 제작 UI다. 프로필, 세계관 설정, 시작 설정, 배경 탭을 제공하고, 커버 이미지와 배경 이미지를 잘라 저장한다.
- 세계관 저장 시 이름 길이, 요약 길이, 프롤로그/초기 상황 길이, 시작 설정 이름을 검증하고 `save_world()`로 전달한다.
- `CharacterEditorPanel`은 썸네일, 메인 성격, 서브 성격, 말투 예시, 외형 설명, 감정별 이미지 4종, 이벤트 이미지를 함께 편집한다.
- 캐릭터 저장 전에는 썸네일과 `neutral/joy/sad/angry` 감정 이미지가 모두 있어야 하며, 메인 성격과 서브 성격 중복도 막는다.
- `ImageCropperPopup`은 드래그와 줌으로 잘라낸 뒤 목표 해상도로 PNG를 저장한다. 캐릭터는 `600x1500`, 세계관 커버는 기본 `1024x1536`, 배경은 `1280x720`에 맞춘다.
- `WorldEditorPanel`도 존재하지만 필드가 단순하고 메인 메뉴에서는 `StoryBuilderPanel`이 직접 연결돼 있어, 현재는 보조 또는 레거시 편집기 성격이 강하다.
